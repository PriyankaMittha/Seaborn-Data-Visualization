# 📊 Seaborn Data Visualization

This repository contains examples, explanations, and practical implementations of the Seaborn library in Python.

Seaborn is a powerful Python visualization library built on top of Matplotlib. It provides a high-level interface for drawing attractive and informative statistical graphics.

---

## 📌 About Seaborn

Seaborn helps in:
- Creating beautiful statistical plots
- Understanding data distribution
- Performing categorical comparisons
- Visualizing relationships between variables
- Creating heatmaps and matrix plots

---

## 📂 Topics Covered

### 1️⃣ Distribution Plots
- histplot()
- kdeplot()
- displot()

### 2️⃣ Categorical Plots
- barplot()
- countplot()
- boxplot()
- violinplot()
- stripplot()
- swarmplot()

### 3️⃣ Relational Plots
- scatterplot()
- lineplot()

### 4️⃣ Regression Plots
- regplot()
- lmplot()

### 5️⃣ Matrix Plots
- heatmap()
- clustermap()

### 6️⃣ Styling & Customization
- Themes
- Color palettes
- Figure size
- Grid styles
